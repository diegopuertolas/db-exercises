# **Ejercicio 3**

Se desea desarrollar un sistema de información para un videoclub que permita gestionar los alquileres de películas por parte de los socios. A continuación, se detallan los requerimientos del sistema:

  - Cada ejemplar está asociado a una única película, pero una película puede tener varios ejemplares disponibles en el videoclub.

  - En cada alquiler, un socio alquila un único ejemplar, pero un ejemplar puede ser alquilado varias veces y por diferentes socios a lo largo del tiempo. Además, un socio puede realizar varios alquileres.

  - Un socio puede realizar varios alquileres, pero cada alquiler es realizado por un único socio.

  - Cada película ha sido dirigida por un único director, pero un director puede haber dirigido varias películas.

  - En cada película, varios actores realizan una interpretación, y cada actor puede interpretar varios papeles en diferentes películas. Además, cada interpretación corresponde a un único actor y a una única película.

  - Una película tiene varias interpretaciones, pero cada interpretación pertenece a una única película.
  
  - Un actor realiza varias interpretaciones, pero cada interpretación es realizada por un único actor.
